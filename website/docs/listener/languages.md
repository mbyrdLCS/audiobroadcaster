# Language & Translation

Audio Broadcaster can translate the service into your language in real time. This is optional — by default you hear the original audio.

## How to Turn On Translation

1. In the listener page, find the **language dropdown**
2. Select your language (Spanish, French, Korean, etc.)
3. On **iPhone/iPad**: tap the **"Tap to Enable Voice"** button that appears
4. On **Android**: selecting a language enables it automatically

When translation is on, you'll hear a voice reading the translated speech in your language.

## Switching Back to Original Audio

Select **"Original"** (or the blank/default option) from the language dropdown. The live audio stream will resume.

## Notes on Translation

- Translation requires the church's internet connection to be working
- There may be a short delay between what is spoken and what you hear translated
- Translation is powered by a free service — if it stops working mid-service, switch back to original audio
